# Agent Roles

This directory holds repo-local guidance for the agent roles described in the
architecture docs. It is documentation-first for now; runtime-specific agent
definitions can be added later once the orchestrator is implemented.

Current role documents:

- `author.md` for spec drafting
- `critic.md` for adversarial review
- `implementer.md` for production code work
- `reviewer.md` for integration and consistency review

These files are intentionally provider-agnostic and should be treated as role
guidance, not as executable configuration.
